# Demacia_Lib 5635

ספריית התשתיות של קבוצה Demacia 5635 לפיתוח רובוטים ב-FRC.

### ❓ מה זה?
זוהי מעטפת שנועדה לפשט את הכתיבה של קוד הרובוט. היא כוללת:

* **קוד למנועים - Talon ו Neo** - הגדרה פשוטה, המרות קלות, הוספה אוטומטית של נתונים ואפשרויות כיול ל-Elastic, שימוש נוח בSysid ועוד!
* **קוד לכל החיישנים הנפוצים ב-FRC** (UltraSonic, Optic, Limit Switch, Pigeon, Cancoder, Encoder)
* **קוד לשאסי**
* **תשתית למנגנונים**
* **טלמטריה - הוספת לוגים**
* **Lookup Table**
* **תשתיות לשלטי XBOX ו PS5**
* **לדים**
